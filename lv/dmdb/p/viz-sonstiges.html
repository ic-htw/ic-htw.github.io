Balkendiagramm - falsch verwendet

  • Balkendiagramm startet nicht bei 0
  • Verzerrte Darstellung
  • Unterschiede erscheinen größer, als sie sind
  • Richtige Verwendung: Start bei 0
  • Unverzerrte Darstellung der Unterschiede
{% include padding-id.html id=3 %}

Linien-/Flächendiagramm - falsch verwendet

  • Linien-/Flächendiagramm startet nicht bei 0
  • Verzerrte Darstellung
  • Unterschiede erscheinen größer, als sie sind
  • Richtige Verwendung: Start bei 0
  • Unverzerrte Darstellung der Unterschiede
{% include padding-id.html id=4 %}

Flächeorientierte Visualisierung

  • Treemap ist quadratische Version von Kuchendiagrammen
  • Balkendiagramm verdeutlicht die Unterschiede besser
