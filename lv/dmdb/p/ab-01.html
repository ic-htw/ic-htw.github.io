Arbeitsblatt 1

Die Abnahme der Ergebnisse erfolgt interaktiv in der Übung

  • Bilden sie Arbeitsgruppen
  • Erstellen sie relationale Datenmodelle mit dem sqldeveloper
  • Wählen sie eines der folgenden Beispiele
    • Bestellung -> Kunde
    • Musiker -> Album
    • Buch -> Autor
    • Produkt -> Hersteller
  • Erstellen sie das Modell mit Fremdschlüsselvariante 1
  • Erzeugen sie den Datenbankcode
  • Legen sie die Tabellen in der Datenbank an
  • Erstellen sie eine Datei mit DML-Befehlen
    • Mehrere Insert-Befehle für jede Tabelle
    • Einen Update-Befehl für jede Tabelle
    • Einen Delete-Befehl für jede Tabelle
    • xxx
  • Erstellen sie das Modell mit Fremdschlüsselvariante 3
  • Hängen sie eine 3 an die Tabellennamen an
    (damit es keine Konflikte gibt)
  • Legen sie die Tabellen in der Datenbank an
  • Wenden sie ihr DML-Befehle auf die neuen Tabellen an
