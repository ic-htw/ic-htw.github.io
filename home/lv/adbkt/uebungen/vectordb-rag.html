Übung Vektordatenbanken - RAG

Erstellen sie eine Anwendung, die Fragen zum Studium der Wirtschaftsinformatik beantworten kann. Diese soll eine Vektordatenbank für RAG nutzen, um Informationen zu speichern und abzufragen.

Beachten sie folgende Punkte

  • Auswahl einer Vektordatenbank
  • Scraping von Texten der WI-Website
  • Zerlegen der Informationen in Chunks
  • Erzeugen von Embeddings
  • Speichern der Embeddings in der Vektordatenbank
  • Hybride Abfragen auf der Vektordatenbank
  • Prompterstellung für ein LLM
  • Prompt an das LLM senden
  • Ergebnis anzeigen
