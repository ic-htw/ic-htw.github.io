Anforderungen an DB-Systeme

Analysieren sie die Anforderungen an DB-Systeme, die sich aus einem Anwendungsgebiet ergeben.

Nutzen sie dazu eine KI und starten sie mit folgenden Prompt

Welche Anforderungen gibt es an Datenbanksysteme 
für das Anwendungsgebiet "aaa". 
Betrachte dabei die folgenden Aspekte:
- Datenmodell
- Konsistenz
- Verfügbarkeit
- Skalierbarkeit
- Performance
- Sicherheit
- Verteilung

Welche Datenbanksysteme haben Kunden-Fallbeispiele für dieses
Anwendungsgebiet veröffentlicht. Wähle 3 Fallbeispiele aus.
Beschreibe kurz das jeweilige Fallbeispiel, gib den Link auf das Fallbeispiel an
und beschreibe, welche Eigenschaften des Datenbanksystems besonders wichtig
zur Erreichung der Ziele war.

Folgende Anwendungsgebiete sollen analysiert werden:

  1. Zahlungsverkehr
  2. Soziale Medien
  3. Flugverkehr
  4. Online-Spiele
  5. Vitaldatenüberwachung von ambulanten Patienten
  6. Aktienhandel
  7. Lieferkettenverwaltung
  8. Produktionssteuerung
  9. Internethandel
  10. Versicherungen
  11. Banken

Jede Gruppe übernimmt den Anwendungsbereich seiner Gruppennummer. Präsentieren sie ihre Ergebnisse.
