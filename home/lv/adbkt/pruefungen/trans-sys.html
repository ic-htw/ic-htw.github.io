Transaktionen in ausgewählten Systemen

Jede Gruppe wählt ein relationales System aus:

  • MySql
  • MariaDB
  • SQL-Server
  • Oracle

Erstellen sie eine Präsentation, die sich an dem Foliensatz "Transaktionskonzepte in Postgres" (link) orientiert:

  • Erstellen sie einen Container für ihr ausgewähltes System
  • Installieren sie den Python-Treiber für ihr System
  • Importieren sie folgendes Notebook in ihren Python-Container (ipynb)(render)
  • Implementieren sie die Szenarien aus Postgres in ihrem System
  • Passen sie den vorgegebenen Python-Code dabei ggfs. an
  • Dokumentieren sie die Ergebnisse
  • Stellen sie insbesondere unterschiedliches Verhalten zu Postgres prägnant dar
