Alle Knoten zu einem Label

MATCH (h:Haltestelle)
RETURN h as haltestelle;
{% include padding-id.html id=2 %}

Matching mit Bedingung

Bedingung mit WHERE

MATCH (h:Haltestelle WHERE h.name STARTS WITH 'H')
RETURN h as haltestelle;

Bedingung als fester Eigenschaftswert

MATCH (h:Haltestelle {name: 'HeidelbergerPlatz'})
RETURN h as haltestelle;
{% include padding-id.html id=3 %}

Knoten sind JSON-Objekte

MATCH (h:Haltestelle {name: 'HeidelbergerPlatz'})
RETURN h as haltestelle;
{% include padding-id.html id=4 %}

Tabellarische Ausgabe von Eigenschaften

MATCH (h:Haltestelle WHERE h.name STARTS WITH 'H')
RETURN h.name as haltestelle
ORDER BY h.name;
{% include padding-id.html id=5 %}

Aggregation

MATCH (h:Haltestelle)
RETURN count(*);
MATCH (h:Haltestelle)
RETURN collect(h.name) as haltestellen;
