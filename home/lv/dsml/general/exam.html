General Remarks

  • The rated exercises dealt with practical skills
  • The lectures had also many hands-on classroom exercises
  • In contrast, the exam checks conceptual understanding
  • It is paper-based, no computer
  • Closed book, no notes allowed
  • Duration: 90 minutes

Exercise 1: Python (6P)

  • Understanding of control statement: assignment, if, loop
  • Understanding of data structures: list, dictionary

You will get program code of a simple program and have to figure out its effect

Exercise 2: Visualization (4P)

  • Understanding, given a use case, which chart type (e.g bar chart, line chart, pie chart) is appropriate
  • Ability to provide examples

Exercise 3: Machine Learning (16P)

  • Understanding Metrics
  • Understanding Decision Trees
  • Understanding how to train models, see (link)

Exercise 4: SQL (18P)

  • Understanding concepts of SQL like:
    • Inner and outer join
    • Aggregation and grouping
    • Subqueries
    • Common table expression (CTE)
    • Window functions

There are two sub parts

  1. You have to write a SQL query based on given scenario
  2. You will be given a bunch of tables and a SQL query and have to figure out its effect

Exercise 5: Business Intelligence (6P)

  • Understanding Multidimensional Data Models
  • Understanding of Star Schema

You will be asked to create a star schema for a given business scenario.
