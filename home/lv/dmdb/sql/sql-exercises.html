Verbund Playlist, Genre, Stueck

{% if page.hide == 0 %} 
SELECT 
  name AS playlist, 
  bezeichnung AS genre, 
  titel
FROM playlist pl
     JOIN stueck s ON s.plid=pl.plid
     JOIN genre g ON g.gid=s.gid
ORDER BY playlist, genre, titel;

Projektkosten pro Mitarbeiter

Kostenansatz:

  • 40 Stunden pro Woche
  • Projekt dauert genau ein Jahr
  • Nur Arbeitskosten, keine Materialkosten
  • Nur Gehalt betrachten, nicht Bonus
{% if page.hide == 0 %} 
SELECT 
  titel, name, 
  stdprowoche/40*gehalt AS kosten
FROM mitarbeiter m
     JOIN maproj mp ON m.mid=mp.mid
     JOIN projekt p ON p.pid=mp.pid
ORDER BY titel, name;

{% endif %}

Wer verdient mehr als sein/ihr Boss

{% if page.hide == 0 %} 
SELECT 
  m.name AS mname, m.gehalt AS mgehalt,
  b.name AS bname, b.gehalt AS bgehalt
FROM mitarbeiter m
     JOIN orgeinheit oe ON oe.oeid=m.oeid
     JOIN mitarbeiter b ON b.mid=oe.leitung
WHERE m.gehalt>b.gehalt
ORDER BY mname;
